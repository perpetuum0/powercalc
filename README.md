# Powercalc

## Power consumption calculator app, written as a Hackathon project

<br>

### Installation

```bash
git clone https://github.com/perpetuum0/powercalc.git
cd powercalc
npm install
```

### Running Development Build

```bash
npm start
```

### Packaging

```bash
npm run package
```
